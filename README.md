# How to run

## Data
`cd .\data`

You'll need to install: pandas, BeautifulSoup, numpy, requests
`python .\refresh-data.py`

## Backend
`cd .\backend`

`npm i`
`npm run dev`

## Frontend
`cd .\frontend`

`npm i`
`npm run start`
